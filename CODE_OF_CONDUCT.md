# Code of Conduct

ZGI is an open-source project for a global technical community.

## Our Standard

We expect project participants to:

- be respectful and constructive
- focus criticism on ideas, code, and outcomes
- assume good intent while still being clear about technical problems
- avoid harassment, personal attacks, discrimination, and intimidation
- avoid publishing private information without explicit permission

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, documentation, community channels, and events connected to the project.

## Enforcement

Maintainers may remove comments, close issues, restrict participation, or take other reasonable action when behavior harms the project community.

For security-sensitive reports, follow `SECURITY.md`. For conduct concerns, contact the maintainers privately through the project’s GitHub repository.
