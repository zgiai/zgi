# Marketing Material

## Use When

Use for marketing visuals, campaign assets, social media images, promotional banners, and commercial creative drafts.

## Payload Rules

- Prefer `generate_image` unless a reference image is provided.
- Include audience, offer, mood, brand-safe color direction, and platform context in the prompt.
- Avoid generating exact brand marks unless the user owns or authorizes them.
- Add a commercial-use reminder in the final answer.

## Data Rules

- Do not claim brand compliance or copyright clearance.
- Avoid misleading before/after claims, fake endorsements, official seals, or deceptive product claims.
- If the user requests text in the image, keep it short and warn that generated text may be imperfect.

## Clarification Rules

Ask when the brand, campaign, target audience, or channel is unclear and would materially change the visual.
