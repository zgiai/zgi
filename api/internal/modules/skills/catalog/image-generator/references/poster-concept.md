# Poster Concept

## Use When

Use for event poster drafts, campaign key visuals, cover concepts, article covers, and presentation cover inspiration.

## Payload Rules

- Prefer `9:16` for mobile posters and `16:9` for article or presentation covers unless the user says otherwise.
- Include event theme, subject, hierarchy, visual mood, and negative space for text.
- Do not rely on the model to render exact readable text.

## Data Rules

- Generated posters are visual drafts, not final production artwork.
- Mention that final typography and brand assets should be checked or added in a design tool.

## Clarification Rules

Ask when the poster size, event theme, or required visual focus is missing.
