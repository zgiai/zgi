# Style Guide

## Use When

Use when the user asks for a visual style or the style is the main missing decision.

## Style Values

- `auto`: Let the model choose based on the prompt.
- `realistic`: Photo-like, natural lighting, real materials.
- `illustration`: Drawn or editorial illustration.
- `flat`: Flat vector-like visual, simple shapes.
- `3d`: Rendered 3D object or scene.
- `guofeng`: Chinese-inspired aesthetic, ink, traditional palette, or cultural visual language.
- `tech`: Futuristic, digital, clean high-tech visual.
- `poster`: Strong key visual suitable for poster drafts.
- `product`: Product-focused concept or ecommerce visual.
- `icon`: Simple icon or app-symbol draft.
- `cover`: Cover-style composition with room for title text.

## Rules

- Do not imitate a living artist's exact style.
- For Chinese output, explain style choices in Chinese.
- When style is missing and important, ask before generating.
