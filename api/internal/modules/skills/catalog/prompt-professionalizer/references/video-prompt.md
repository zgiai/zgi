# Video Prompt

## Use When

Use for prompts targeting animation or video generation tools such as 可灵, Runway, Pika, 即梦视频, Sora-style tools, or similar text-to-video tools.

## Required Elements

Check for:

- Subject: main character, product, object, environment, or abstract element.
- Scene: location, background, time, weather, atmosphere.
- Action: what changes over time, what the subject does, what moves.
- Camera movement: push in, pull out, pan, tilt, orbit, handheld, static, tracking shot.
- Duration: seconds or rough length.
- Rhythm: slow, fast, cinematic, smooth, energetic, calm.
- Style: realistic, 3D, anime, commercial, documentary, product film, sci-fi, ink, flat.
- Lighting and color: mood, contrast, time of day, palette.
- Transition: cut, fade, match cut, scene change, no transition.
- Prohibited content: unwanted motion, flicker, distortion, extra limbs, text artifacts, logo misuse.

## Prompt Shape

Write the prompt as:

- Opening shot.
- Subject and environment.
- Main action.
- Camera movement.
- Mood, lighting, and style.
- Duration and rhythm.
- Negative prompt or forbidden elements.

## Clarification Rules

Ask when action or duration is missing. If camera movement is missing, use a conservative default such as slow push-in and mark it under `默认假设`.

## Boundary Rules

- Do not create prompts for dangerous instructions, sexual content, graphic violence, deceptive identity, or infringing imitation.
- Do not promise physical accuracy, exact likeness, or perfect temporal consistency unless the target tool supports it and the user provided references.
