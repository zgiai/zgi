# Image Prompt

## Use When

Use for prompts targeting image generation tools such as Midjourney, Stable Diffusion, DALL-E, 即梦, 可灵, 通义万相, or other text-to-image tools.

## Required Elements

Check for:

- Subject: person, product, object, scene, brand-safe concept, or abstract visual.
- Scene: environment, location, background, time of day, season, atmosphere.
- Style: realistic, illustration, flat, 3D, Chinese style, technology, commercial poster, editorial, product render.
- Composition: main subject placement, foreground/background, negative space, symmetry, focus.
- Lighting: natural, studio, cinematic, neon, soft, high contrast.
- Color: dominant palette, brand colors, mood colors.
- Material and detail: glass, metal, fabric, skin, paper, screen, texture, quality level.
- Camera/view: close-up, wide shot, top view, isometric, macro, portrait, product shot.
- Aspect ratio or size: 1:1, 16:9, 9:16, 4:3, poster, banner, cover.
- Negative prompt: unwanted objects, artifacts, text errors, low quality, distorted hands, watermark, logo misuse.

## Prompt Shape

Include:

- Main visual description.
- Style and rendering quality.
- Composition and camera.
- Lighting and color.
- Aspect ratio or output format.
- Negative prompt when useful.

## Clarification Rules

Ask when subject or style is missing. If only aspect ratio is missing, use a default and mark it under `默认假设` unless the final use case makes ratio critical.

## Boundary Rules

- Do not request imitation of a specific living artist or living person's likeness.
- Do not create prompts for illegal, pornographic, graphic violent, counterfeit, or deceptive identity imagery.
- Do not claim commercial clearance for generated images.
