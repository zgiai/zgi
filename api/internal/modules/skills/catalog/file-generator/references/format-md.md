# Markdown Reference

Use this reference when `format` is `md` or `markdown`.

## Tool Parameters

- `content`: Markdown source text.
- `format`: Use `md`.
- `filename`: Optional display filename without path separators. The `.md` extension is added automatically.
- `title`: Ignored for Markdown.
- `lifecycle`: Optional `persistent` or `temporary`. Defaults to `persistent`.

## Content Rules

- Provide valid Markdown source.
- Use Markdown for reports, notes, plans, structured prose, README-style content, and documentation drafts.
- The tool saves Markdown source as-is. It does not render Markdown into HTML, DOCX, or PDF styling.
