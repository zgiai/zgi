# Contributing To ZGI Web

ZGI is maintained as a monorepo. Use the repository-level contribution guide for pull requests, issue reporting, commit format, and review expectations:

- [../CONTRIBUTING.md](../CONTRIBUTING.md)

For web-only changes, run the relevant checks from this directory before opening a pull request:

```bash
pnpm lint
pnpm type-check
pnpm build
```

Contributions are licensed under the repository license unless a file states otherwise.
